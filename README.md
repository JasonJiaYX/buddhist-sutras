# 🪷 禅意阅经系统 (Zen Sutra Reader)

一个极简、优雅且极具禅意的国学与佛经阅览系统。专为静心阅读、对照研习传统经典而设计。

🌐 **在线演示网址**：[https://jasonjiayx.github.io/buddhist-sutras/](https://jasonjiayx.github.io/buddhist-sutras/)

---

## 🌟 核心特性

- 📖 **多元经典集成**：收录大乘佛经（《心经》、《金刚经》、《大悲咒》、《普门品》）、大乘论典（《中论》）以及儒道经典（《道德经》、《论语》、《中庸》）。
- ☯️ **极速译注对照**：
  - 对《论语》、《中庸》等核心经典提供现代白话文翻译和详细的字词注释。
  - 支持“译”、“注”独立开关。页面采用极速 CSS 类映射机制，**无刷新、瞬时切换显隐**，拒绝页面卡顿。
- 🎨 **禅意美学视觉**：
  - 提供多套精心调配的 HSL 柔和主题（如古朴竹简、清雅沙滩、水墨淡黑等），呵护双眼。
  - 支持 **14px 到 32px 字号自由无级微调**，伴随行高比例自适应优化，带来极佳的阅读排版。
- 🐟 **电子木鱼静心**：
  - 纯 Web Audio API 底层波形合成的空心回声木鱼音效，无需加载任何音频文件，体积轻量。
  - 支持一键开启“自动敲击”模式，伴随“功德+1”、“烦恼-1”、“智慧+1”等 6 种随机漂浮字效，累积功德。
- 🔖 **段落精细书签**：
  - 点击任意段落或小章，即可瞬间添加/取消书签。
  - 侧边栏书签抽屉提供优雅的管理面板，支持一键平滑滚动并闪烁高亮定位到指定段落。

---

## 🛠️ 技术栈

- **前端核心**：HTML5, Vanilla CSS3 (原生 CSS 变量自适应), JavaScript (ES6+)
- **音效合成**：Web Audio API (OscillatorNode 弹性频率三角波形合成)
- **数据管理**：结构化 JSON 数据库，基于 Python 爬虫自动清洗、解析并编译生成。
- **字体方案**：引入 Google Fonts 汉字衬线体 `Noto Serif SC`（思源宋体），体现典雅的排版美学。

## 🚀 本地快速启动

由于浏览器处于安全考虑会拦截本地静态网页获取 JSON 数据库文件的请求（跨域拦截），建议使用以下任一简单方式启动本地预览：

### 方法一：使用 Python 启动微型服务器（推荐）
1. 在项目根目录下打开终端（PowerShell 或 CMD）。
2. 执行以下命令：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

### 方法二：使用 VS Code Live Server 插件
如果您使用 VS Code 编辑器，可以直接在右下角点击 **"Go Live"** 即可一键启动预览。

- `lunyu.json` - 论语 (杨伯峻译注版，完整二十篇)
- `zhong-yong.json` - 中庸 (权威译注版，完整三十三章)
