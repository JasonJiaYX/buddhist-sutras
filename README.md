# 🪷 禅意阅经与国学每日推送系统 (Zen Classics Reader & Pusher)

一个极简、优雅且极具禅意的国学经典阅览与每日自动推送系统。专为静心阅读、对照研习传统经典，以及在日常生活中体悟国学智慧而设计。

🌐 **在线演示网址**：[https://jasonjiayx.github.io/buddhist-sutras/](https://jasonjiayx.github.io/buddhist-sutras/)

---

## 🌟 核心特性

- 📖 **多元经典集成**：
  - **佛家经典**：《心经》、《金刚经》（完整 32 分）、《大悲咒》、《普门品》、《中论》（完整四卷 27 品）、《六祖坛经》（完整 10 品）。
  - **道家经典**：《道德经》（完整 81 章）、《太上老君说常清静经》、《庄子》（南华真经，完整内/外/杂篇共 33 篇）。
  - **儒家经典**：《论语》（精校译注版）、《中庸》（精校译注版）、《周易》（完整 64 卦卦爻辞与大象传，以及《系辞传》上下、《说卦传》、《序卦传》、《杂卦传》等全部“十翼”传文，完美集成 Unicode 卦象符号 `䷀`~`䷁`）。
  - **修身劝善**：《了凡四训》（完整四篇）、《太上感应篇》、《文昌帝君阴骘文》。
- ☯️ **极速译注对照**：
  - 对《论语》、《中庸》等核心经典提供现代白话文翻译和详细的字词注释。
  - **单击展开/折叠**：默认折叠全部译注以还原纯净的原文版面，单击任意段落即可顺滑展开/折叠该段译注。
- 🎨 **禅意美学视觉**：
  - 提供多套精心调配的 HSL 柔和主题（如古朴竹简、清雅沙滩、水墨淡黑等），自适应行高与字号自由微调。
  - **朱批与金粉写经（书签状态）**：双击任意段落即可将其收藏，在浅色主题下该段原文呈现典雅的**朱砂红**（模拟古籍朱批），在深色主题下则呈现**亮金色**（模拟泥金写书），并伴有温润背景晕染。
- 🐟 **电子木鱼静心**：
  - 纯 Web Audio API 底层波形合成 of 空心回声木鱼音效，支持一键开启“自动敲击”模式，伴随 6 种随机漂浮字效，累积功德。
- 🔖 **书签抽屉导航**：
  - 双击段落无痕收藏后，可在侧边栏书签抽屉中集中管理，支持一键平滑滚动并闪烁高亮定位到指定段落。
- 🤖 **每日国学飞书推送 (Feishu Pusher)**：
  - **智能推送机制**：通过 Python 脚本 `push_daily_classics.py`，每日随机从经典库中选择章节与文段。
  - **首创“大师级”AI 解读**：拒绝肤浅的鸡汤，针对不同书籍自动定制相应的注疏流派（如《周易》适配王弼与程颐、《庄子》适配郭象与南怀瑾、《论语》适配朱熹与钱穆、《坛经》适配憨山大师）。AI 会自动生成包括译文、宗师视角的思想深度解读、以及日常修行实践指引的精美飞书卡片。
  - **公农历日期呈现**：每日推送自动带上当前的公历日期与精确的农历干支日期（由 `zhdate` 库计算）。
  - **自动化云部署**：通过 GitHub Actions 工作流在每日北京时间 **07:27** 准时运行并推送到飞书指定群聊。

## 🛠️ 技术栈

- **前端核心**：HTML5, Vanilla CSS3 (原生 CSS 变量自适应), JavaScript (ES6+)
- **音效合成**：Web Audio API (OscillatorNode 弹性频率三角波形合成)
- **数据管理**：结构化 JSON 数据库，基于 Python 爬虫自动清洗、解析并编译生成。
- **字体方案**：引入 Google Fonts 汉字衬线体 `Noto Serif SC`（思源宋体），体现典雅的排版美学。
- **推送服务**：Python 3, `requests`, `zhdate` 农历库, Gemini API (大语言模型解析), GitHub Actions, Lark/Feishu API.

## 🚀 本地快速启动

### 网页阅读端
由于浏览器出于安全考虑会拦截本地静态网页获取 JSON 数据库文件的请求（跨域拦截），建议使用以下任一简单方式启动本地预览：

#### 方式一：使用 Python 启动微型服务器（推荐）
1. 在项目根目录下打开终端（PowerShell 或 CMD）。
2. 执行以下命令：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

#### 方式二：使用 VS Code Live Server 插件
如果您使用 VS Code 编辑器，可以直接在右下角点击 **"Go Live"** 即可一键启动预览。

### 每日推送端 (Feishu Pusher)
1. 安装依赖：
   ```bash
   pip install requests zhdate
   ```
2. 在项目根目录下创建本地 `config.json` 文件（此文件已加入 `.gitignore` 保护隐私）：
   ```json
   {
     "GEMINI_API_KEY": "您的Gemini_API_Key",
     "FEISHU_WEBHOOK": "您的飞书自定义机器人Webhook"
   }
   ```
3. 本地运行测试：
   ```bash
   python push_daily_classics.py
   ```
